# Architecture Decision Records

This directory records significant technical decisions for DevKit.

Use the following template for new decisions:

```markdown
# ADR-NNN: Decision title

- Status: Proposed | Accepted | Superseded
- Date: YYYY-MM-DD

## Context

What problem or constraint prompted the decision?

## Decision

What approach was selected?

## Consequences

What trade-offs and follow-up work result from the decision?
```

Initial decisions to document as the project evolves include API boundaries,
authentication, database schema ownership, and frontend deployment.
